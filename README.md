# HackerNews Upvote Prediction

Predicting which Hacker News posts will go viral using machine learning. Built on a dataset of ~3.9M posts spanning 2006–2023.

## Overview

This project explores what makes content succeed on Hacker News by training models to predict post scores. It includes content extraction from linked articles, custom-trained embeddings, and various prediction models — from classical baselines to neural approaches.

## Setup

```bash
pip install -r requirements.txt
```

## Dataset

The project uses a publicly available Hacker News dataset (`hn.csv`) with ~3.9M posts. The dataset is not included in this repo due to size.
